对象的定义

函数也是一个对象
函数就是可以重复执行的代码块
函数就是完成特定功能的一段代码
使用<span class="hljs-keyword">typeof</span>检查一个函数对象时，会返回<span class="hljs-function"><span class="hljs-keyword">function</span>
</span>`</pre>

### 作用

<pre class="hljs undefined">`因为一部分代码使用次数可能会很多，所以封装起来，需要的时候调用就可以了。
1)将复杂的事情简单化
2)将重复使用的代码进行统一封装

特点:
1)封装到函数中的代码不会立即执行
2)函数中的代码会在函数调用的时候执行
3)调用函数 语法：函数对象()
4)调用函数时，函数中封装的代码会按照顺序执行
`</pre>

### 函数的参数

<pre class="hljs javascript">`为什么要设置参数？
为了增强函数的功能性和函数的可拓展性，便于交互。

形参:
<span class="hljs-number">1</span>)形式上参与运算的变量，无实际值，为实参占位置，就像一个躯壳一样。
<span class="hljs-number">2</span>)<span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">f</span>(<span class="hljs-params">a,b</span>)</span>{}  <span class="hljs-comment">//a,b是形参，占位用，函数定义时形参无值</span>

实参:
<span class="hljs-number">1</span>)实际参与运算的变量。形参为他占位置，真实参与运算的变量。
<span class="hljs-number">2</span>)f(x, y); <span class="hljs-comment">//x, y实参，有具体的值，会把x,  y复制一份给函数内部的a和b，函数内部的值是复制的新值，无法修改外部的x,y</span>

注意:在其它语言中实参个数必须和形参个数一致，
但是JavaScript中没有函数签名的概念，实参个数和形参个数可以不相等

      </div>

函数使用注意

<span class="hljs-number">1.</span> 在调用函数时，可以在()中指定实参,  实参将会赋值给函数中对应的形参

<span class="hljs-number">2.</span> 调用函数时解析器不会检查实参的类型,   所以开发中一般需要对参数进行类型的检查

<span class="hljs-number">3.</span> 函数的实参可以是任意的数据类型

<span class="hljs-number">4.</span> 调用函数时，解析器不会检查实参的数量, 多余实参不会被赋值, 

   如果实参的数量少于形参的数量，则没有对应实参的形参将是<span class="hljs-literal">undefined</span>
`</pre>

### 函数返回值（return）

<pre class="hljs javascript">`<span class="hljs-number">1</span>)当一个函数被调用，通常会从函数的开始执行到结束。

<span class="hljs-number">2</span>)如果想提前结束该函数的执行可以使用<span class="hljs-keyword">return</span>语句，<span class="hljs-keyword">return</span>语句后面的所有语句将永远不会执行。

<span class="hljs-number">3</span>)一般<span class="hljs-keyword">return</span>用于返回结果。

<span class="hljs-number">4</span>)注意
  如果函数没有显示的使用 <span class="hljs-keyword">return</span>语句 ，那么函数有默认的返回值：<span class="hljs-literal">undefined</span>

  如果函数使用 <span class="hljs-keyword">return</span>语句，那么跟再<span class="hljs-keyword">return</span>后面的值，就成了函数的返回值

  如果函数使用 <span class="hljs-keyword">return</span>语句，但是<span class="hljs-keyword">return</span>后面没有任何值，那么函数的返回值也是：<span class="hljs-literal">undefined</span>

  推荐的做法是要么让函数始终都返回一个值，要么永远都不要返回值。
`</pre>
