JavaScript之正则表达式

2019-05-03

1. 定义

正则表达式用于定义一些字符串的规则，计算机可以根据正则表达式，
来检查一个字符串是否符合规则，或者将字符串中符合规则的内容提取出来
`</pre>

2. 举例

<pre class="hljs css">`邮箱的验证
<span class="hljs-selector-tag">yjh</span>@<span class="hljs-keyword">itlike</span>.<span class="hljs-keyword">com</span>

规则
<span class="hljs-number">1</span>.前边可以是任意的
<span class="hljs-number">2</span>.跟着一个@
<span class="hljs-number">3</span>.后边可以是任意的
<span class="hljs-number">4</span>.跟着一个
<span class="hljs-number">5</span>.后边可以是任意的
`</pre>

4. 字面量模式使用

<pre class="hljs javascript">`<span class="hljs-number">1.</span> 定义

<span class="hljs-keyword">var</span> reg = <span class="hljs-regexp">/a/i</span>;

<span class="hljs-number">3.</span> 创建一个正则表达式检查一个字符串中是否有字母
[]里的内容也是或的关系
[ab] == a|b

[A-Z] 任意大写字母
[A-z] 任意字母
[<span class="hljs-number">0</span><span class="hljs-number">-9</span>] 任意数字

<span class="hljs-number">4.</span> 量词
通过量词可以设置一个内容出现的次数

量词只对它前边的一个内容起作用
{n} 正好出现n次
{m,n} 出现m-n次
{m,} m次以上
+ 至少一个，相当于{<span class="hljs-number">1</span>,}
* <span class="hljs-number">0</span>个或多个，相当于{<span class="hljs-number">0</span>,}
? <span class="hljs-number">0</span>个或<span class="hljs-number">1</span>个，相当于{<span class="hljs-number">0</span>,<span class="hljs-number">1</span>}

<span class="hljs-number">5.</span> 匹配开头和结尾
检查一个字符串中是否以某个字符开头
^ 表示开头
$ 表示结尾

如果在正则表达式中同时使用^ $则要求字符串必须完全符合正则表达式

<span class="hljs-number">6.</span> 检查一个字符串是否是一个合法手机号
